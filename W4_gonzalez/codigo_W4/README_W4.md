# Ejecución del Trabajo 4

Este directorio contiene los scripts para procesar, auditar y modelar el dataset.
Deben ejecutarse en orden desde la raíz del proyecto (trabajofinal).

## Orden de Ejecución

1. `python W4_gonzalez/codigo_W4/00_setup.py`: Carga, deduplica y particiona los datos.
2. `python W4_gonzalez/codigo_W4/01_auditoria_calidad.py`: Analiza faltantes y calidades.
3. `python W4_gonzalez/codigo_W4/02_representacion.py`: Estudia y transforma las representaciones.
4. `python W4_gonzalez/codigo_W4/03_pipelines.py`: Compara el modelo ingenuo con el coherente.
5. `python W4_gonzalez/codigo_W4/04_resultados.py`: Sintetiza la bitácora final.
